Level 1.1 Line

Task Draw a line by calling polarLine() function. See the function details below.

Hint polarLine() is the function to draw a single line. Three parameters can be passed to the single drawing funciton by the following order: angle, radius, and distance (distance is an optional parameter). For example, polarLine(30, 50, 10) draws a line starts with 30 degree of angle, radius of 50, and 10 distance from center point.
